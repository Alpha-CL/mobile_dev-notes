#### mobile_event vs pc_event

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


// mobile_event vs pc_event


1) 出发顺序

    touchstart -> touchend -> mousedown -> click -> mouseup
    
    ( pc_event 在移动端中会有延迟 "300ms左右" )
    
    * 07 年iphone 发布时，手机无适配的概念，为了解决手机上 展示 pc 网页太小
      ios 添加双击放大，再次双击还原 的功能，但若点击的时候为链接，则可能错误跳转
      则为手机事件触发延迟 300ms


2) touchstart vs click

    - touchstart: 手指触碰到元素即出发
    
    - click: 手指触碰到元素并抬起才会出发


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
